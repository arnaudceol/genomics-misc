genomics-misc
=============

Misc libraries, methods and apps for managing genomics data. Most of it is about sharing pieces of code from other projects that have not been released yer

Build the package:
> mvn clean install appassembler:assemble assembly:assembly

A directory called CRUGenomics will be created in the directory target/. It can be copied somewhere else.

To run a program, go to CRUGenomics and run 
> ./bin/program -h (Linux/MaxOS)
> bin/program.exe -h  (Windows)
